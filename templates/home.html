8/18 A humbling lesson

I wanted to create an implementation of Dijkstra's algorithm for map routing, which I have done before, and it didn't go well. I dove straight into it, and wrote all of the code without running and checking anything. Of course that didn't go well. Of course. I knew that it wouldn't work well if I did it that way.

And I ignored it anyway.

So next time I'll properly write and test everythin. My code failed at the first line. I couldn't import from another file. I also didn't really know where I was headed most of the time, or what data structures I should be using. It was terrible, a mess, but a good lesson learned.

So I won't rush through things. So I won't be so ambitious straight away.
